# Project Context

Interactive educational web platform / scrollytelling creative tool.

## Domain
Physics simulation (Rapier.js WASM), DOM-free text layout (Pretext), and 2D canvas animation (ragdoll stick figures).

## Stakeholders

- **Downstream consumers:** Spec Kit. Once a feature is approved, Strata should output a Product Brief that can be directly used as the input for a `/speckit.specify` command.
